
# English Dungeon Adventure

## Product Requirements Document (PRD)

Version: 1.0

---

# 1. Vision

English Dungeon Adventure is a gamified English-learning application designed for an 11-year-old Spanish-speaking student.

The goal is to reinforce and consolidate all English grammar, vocabulary and structures learned during the previous academic year while keeping motivation high throughout the summer.

The application must feel like an adventure game first and an educational platform second.

The player should want to come back because it is fun, rewarding and challenging.

---

# 2. Primary Goal

At the end of the summer the student should:

* Retain previously learned English knowledge.
* Improve confidence using English.
* Arrive fully prepared for Class 5.
* Associate English practice with positive experiences.

---

# 5. Learning Content

The application must be based exclusively on the curriculum supplied by the uploaded academy materials.

No external grammar topics should be introduced unless explicitly approved.

# 10. Difficulty Model

The game should not be easy.

Questions should match the difficulty of the academy materials.

Rules:

* No baby-level English.
* No random guessing.
* No excessive repetition.
* No trick questions.
* Gradually increase difficulty.

Target success rate:

70–85%

---

# 11. Adaptive Learning

Each grammar topic receives a Mastery Score.

Range:

0–100

The system should:

* Repeat weak topics more often.
* Reduce repetition of mastered topics.
* Mix review with new content.

Recommended distribution:

70% current mission

30% review content
